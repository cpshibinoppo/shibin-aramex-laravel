# Laravel Aramex SDK (Modern + Clean)

A clean, modern, senior-level Laravel SDK for Aramex.  
Supports all major Aramex SOAP services:

- Create Shipments  
- Create Pickups  
- Calculate Rate  
- Track Shipments  
- Fetch Countries  
- Fetch Cities  
- Validate Address  

Built with:
- DTO-based requests  
- Automatic WSDL selection (Test / Live)  
- Central SOAP Factory  
- Simple Validator wrapper  
- Facade support  
- Laravel auto-discovery  
- PHP 8+  
- Fully type-safe
